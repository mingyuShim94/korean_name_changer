# Task ID: 1
# Title: Setup Next.js Project with Tailwind CSS and Shadcn UI
# Status: done
# Dependencies: None
# Priority: high
# Description: Initialize a new Next.js project with App Router, configure Tailwind CSS, and set up Shadcn UI components library for the Korean Name Changer application.
# Details:
1. Create a new Next.js project using App Router:
```bash
npx create-next-app@latest korean-name-changer --typescript --tailwind --eslint --app
```
2. Navigate to the project directory:
```bash
cd korean-name-changer
```
3. Install Shadcn UI dependencies:
```bash
npm install shadcn-ui @radix-ui/react-icons class-variance-authority clsx tailwind-merge
```
4. Initialize Shadcn UI:
```bash
npx shadcn-ui@latest init
```
5. Configure the project structure according to Next.js App Router conventions:
   - Create appropriate directories in the app folder
   - Set up layout.tsx with proper metadata
   - Configure theme and global styles
6. Create a components directory for reusable UI components

# Test Strategy:
1. Verify the project builds without errors using `npm run build`
2. Ensure the development server runs correctly with `npm run dev`
3. Confirm Tailwind CSS is working by testing a simple component with Tailwind classes
4. Test that Shadcn UI components can be imported and rendered correctly
5. Verify the responsive design works on different viewport sizes
